# Web 平台使用指南

本文档展示如何使用 AlphaScope 的 Web 平台。

## 安装依赖

```bash
pip install streamlit plotly
```

## 启动应用

```bash
cd /Users/hudoudou/Documents/4_code/1_stock/alphascope
streamlit run src/web/app.py
```

## 功能概览

### 1. 控制面板（左侧边栏）

#### 资金配置
- **初始资金**: 设置回测的初始资金（默认 100 万）

#### 策略参数
- **选择策略**: 选择不同的交易策略
- **短期均线**: 设置短期均线周期（3-20）
- **长期均线**: 设置长期均线周期（10-60）

#### 风控参数
- **止损线**: 设置止损百分比（-20% 到 -1%）
- **止盈线**: 设置止盈百分比（5% 到 50%）

#### 回测区间
- **开始日期**: 选择回测开始日期
- **结束日期**: 选择回测结束日期

#### 启动回测
- 点击"🚀 启动回测"按钮开始回测

---

### 2. 核心指标卡（顶部）

展示关键回测指标：
- **总收益率**: 回测期间的总收益
- **年化收益率**: 年化后的收益率
- **最大回撤**: 最大资产回撤
- **夏普比率**: 风险调整后收益
- **总交易次数**: 交易总次数
- **胜率**: 盈利交易占比

---

### 3. 资产净值与回撤图

#### 资产净值曲线
- 展示策略每日总价值变化
- 归一化处理（从 1.0 开始）
- 支持缩放和悬浮提示

#### 回撤图
- 展示动态回撤百分比
- 高亮最大回撤区间

---

### 4. K线图与交易信号

#### K线图
- 展示股票的开盘、最高、最低、收盘价
- 支持选择不同股票

#### 交易信号
- 在 K 线上标记买入（B）和卖出（S）点
- 显示交易详情

---

### 5. 数据明细

#### 交易明细表
- 交易时间
- 股票代码
- 交易方向（买入/卖出）
- 交易数量
- 交易价格
- 手续费
- 成交额

#### 每日资产状态表
- 日期
- 可用资金
- 持仓市值
- 总资产
- 支持导出 CSV

---

## 使用示例

### 1. 基本使用

```python
# 1. 启动应用
streamlit run src/web/app.py

# 2. 在左侧边栏配置参数
#    - 设置初始资金：1000000
#    - 选择策略：SimpleMAStrategy
#    - 设置均线：5日和20日
#    - 设置止损：-8%
#    - 设置止盈：20%
#    - 选择日期：2024-01-01 到 2024-03-31

# 3. 点击"启动回测"

# 4. 查看结果
#    - 顶部指标卡显示关键指标
#    - 资产曲线图展示净值变化
#    - K线图展示交易信号
#    - 数据表格展示交易明细
```

### 2. 自定义策略

```python
from src.web.app import SimpleMAStrategy

# 创建自定义策略
strategy = SimpleMAStrategy(
    ma_short=10,
    ma_long=30,
)

# 设置风控参数
strategy.stop_loss_pct = -10.0
strategy.take_profit_pct = 30.0
```

## 注意事项

1. **数据准备**: 确保已下载股票数据
2. **参数调整**: 调整参数后需重新点击"启动回测"
3. **异常处理**: 如果无交易信号，会显示友好提示
4. **性能**: 大量数据可能需要较长加载时间
