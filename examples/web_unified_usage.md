统一Web平台使用指南

本文档展示如何使用 AlphaScope 的统一Web平台。

## 安装依赖

```bash
pip install streamlit plotly
```

## 启动应用

```bash
streamlit run src/web/app_unified.py
```

## 功能概览

### 1. 数据下载与同步

#### 下载范围控制

- **单股下载**: 输入单个股票代码
- **批量清单**: 文本输入或文件上传
- **全量下载**: 下载所有A股股票数据

#### 自动数据源切换

系统会自动在以下数据源之间切换：
1. AKShare（默认）
2. BaoStock
3. Tushare

当某个数据源下载失败时，自动切换到下一个数据源。

#### 时间与复权配置

- **日期选择**: 开始日期、结束日期
- **复权类型**: 前复权、后复权、不复权
- **时间频度**: 日线、周线、5分钟线、1分钟线

#### 下载监控

- **动态进度条**: 实时显示下载进度
- **吞吐量面板**: 显示当前下载速度和预计剩余时间
- **实时日志**: 显示下载过程中的详细日志
- **数据源显示**: 显示当前使用的数据源

### 2. 本地库存管理

- 展示已下载的股票列表
- 显示数据范围、记录数、文件大小
- 统计总股票数和总数据量

### 3. K线验真

- **K线图**: Candlestick 图表
- **成交量图**: 柱状图
- **交互功能**: 
  - 鼠标悬浮显示 OHLC 价格
  - 支持缩放和拖动
  - 支持数据质量检查

### 4. 回测分析

- 策略回测
- 指标分析
- 结果可视化

## 数据存储

### 存储结构

```
./data/
├── 600000.SH.parquet      # 原始K线数据
├── 000001.SZ.parquet
├── 300750.SZ.parquet
└── processed/              # 预处理后的数据
    ├── 600000.SH.parquet
    └── ...
```

### 数据格式

所有数据使用 Parquet 格式存储，包含以下字段：

- date: 交易日期
- open_price: 开盘价
- high_price: 最高价
- low_price: 最低价
- close_price: 收盘价
- volume: 成交量
- amount: 成交额
- code: 股票代码

## 自动数据源切换逻辑

1. **默认使用 AKShare**
   - 免费，无需注册
   - 速度快，稳定性好

2. **失败时切换到 BaoStock**
   - 免费，需要登录
   - 数据质量高

3. **最后尝试 Tushare**
   - 需要 Token
   - 数据最全面

4. **所有数据源失败**
   - 记录失败日志
   - 继续下载下一只股票

## 异常处理

### 防重复触发

- 下载任务执行时，所有配置项和按钮自动禁用
- 防止重复点击导致线程冲突

### 容错提示

- 单个股票下载失败不会中断整个任务
- 失败的股票会被记录并显示在日志中
- 下载结束后显示成功/失败统计

### 异步处理

- 下载任务在后台线程执行
- 前端通过轮询获取进度
- 不会阻塞UI渲染

## 注意事项

1. **网络连接**
   - 确保网络连接稳定
   - 长时间等待时检查网络

2. **数据源限制**
   - AKShare: 无限制
   - BaoStock: 需要登录
   - Tushare: 需要 Token

3. **数据存储**
   - 数据存储在 `./data` 目录
   - 使用 Parquet 格式，Snappy 压缩
   - 按股票代码分文件存储
