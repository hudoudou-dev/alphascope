Web应用V2使用指南

本文档展示如何使用 AlphaScope 的数据管理平台。

## 安装依赖

```bash
pip install streamlit plotly
```

## 启动应用

```bash
streamlit run src/web/app_v2.py
```

## 功能概览

### Tab 1: 数据下载与同步工作台

#### 下载范围控制

1. **单股下载**
   - 输入单个股票代码
   - 格式: XXXXXX.EXCHANGE（如 600000.SH）

2. **批量清单**
   - 文本输入: 每行一个代码，或用逗号分隔
   - 文件上传: 支持 .csv/.txt 文件

3. **全量下载**
   - 下载所有A股股票数据
   - 包括主板、创业板、科创板

#### 时间与复权配置

- **日期选择**: 开始日期、结束日期
- **复权类型**: 前复权、后复权、不复权
- **时间频度**: 日线、周线、5分钟线、1分钟线
- **数据源**: AKShare、BaoStock、Tushare

#### 下载监控

- **动态进度条**: 实时显示下载进度
- **吞吐量面板**: 显示当前下载速度和预计剩余时间
- **实时日志**: 显示下载过程中的详细日志

### Tab 2: 本地库存与K线浏览器

#### 本地资产看板

- 展示已下载的股票列表
- 显示数据范围、记录数、文件大小
- 支持搜索和排序

#### K线图预览

- **主图**: K线图（Candlestick）
- **副图**: 成交量柱状图
- **交互功能**: 
  - 鼠标悬浮显示 OHLC 价格
  - 支持缩放和拖动
  - 支持数据质量检查

## 使用示例

### 1. 单股下载

```python
# 1. 选择"单股下载"
# 2. 输入股票代码: 600000.SH
# 3. 选择日期范围: 2024-01-01 到 2024-01-31
# 4. 选择复权类型: 前复权
# 5. 点击"开始下载"
```

### 2. 批量下载

```python

# 2. 选择"文本输入"
# 3. 输入股票代码列表:
#    600000.SH
#    000001.SZ
#    300750.SZ
# 4. 点击"开始下载"
```

### 3. 查看K线图

```python
# 1. 切换到"本地库存与K线浏览器"标签页
# 2. 在表格中选择股票
# 3. 查看K线图和成交量
```

## 异常处理

### 防重复触发

- 下载任务执行时，所有配置项和按钮自动禁用
- 防止重复点击导致线程冲突

### 容错提示

- 单个股票下载失败不会中断整个任务
- 失败的股票会被记录并显示在日志中
- 下载结束后显示成功/失败统计

### 异步处理

- 下载任务在后台线程执行
- 前端通过轮询获取进度
- 不会阻塞UI渲染

## 注意事项

1. **数据源选择**
   - AKShare: 免费，无需注册
   - BaoStock: 免费，需要登录
   - Tushare: 需要Token

2. **下载频率**
   - 避免频繁请求，可能被限流
   - 建议使用"智能更新"功能

3. **数据存储**
   - 数据存储在 `./data` 目录
   - 使用 Parquet 格式，Snappy 压缩
   - 按股票代码分文件存储
