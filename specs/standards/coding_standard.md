# Coding Standard

## Python Version

Python 3.10+

---

## Style

- PEP8
- Ruff
- Black

---

## Naming

- snake_case
- PascalCase
- UPPER_SNAKE_CASE

---

## Typing

所有公共接口必须包含 typing。

---

## Docstring

所有公共函数必须包含 docstring。

---

## Forbidden

禁止：

- mutable global state
- hardcoded config
- silent exception
