# AlphaScope 多维度、多策略组合选股方案

> 基于本地项目现状 + GitHub 主流量化项目 + 学术论文，设计渐进式升级方案

## 一、现状诊断

### 1.1 当前优势

| 维度 | 现状 |
|------|------|
| 5因子体系 | 趋势(35%)+动量(25%)+量能(20%)+波动(10%)+基本面(10%)，已落地 |
| 回调买点 | 识别上升趋势中的回调，非对称评分 |
| 风控 | ST过滤、涨停过滤、止损-15%、止盈+20%、冷却期5天 |
| 回测 | 完整的 BacktestEngine，含夏普比率、最大回撤等指标 |
| 架构 | StrategyCombiner 已预留多策略组合接口 |

### 1.2 核心短板

| 短板 | 影响 | 优先级 |
|------|------|--------|
| **单一策略** | 只有一套5因子评分，牛熊震荡一视同仁，缺乏市场状态自适应 | ⭐⭐⭐ |
| **无行业/板块维度** | 不区分行业特征，科技股和银行股用同一套标准打分 | ⭐⭐⭐ |
| **无资金流向因子** | 缺少北向资金、主力资金、龙虎榜等机构行为信号 | ⭐⭐⭐ |
| **无市场状态识别** | 没有牛熊/震荡判断，策略参数一成不变 | ⭐⭐ |
| **因子数量偏少** | 仅~10个因子，远低于 Qlib Alpha158(158个) 和 WorldQuant 101(101个) | ⭐⭐ |
| **无AI/ML能力** | 纯规则打分，没有机器学习排序或预测 | ⭐⭐ |
| **基本面数据弱** | PE/PB/ROE 数据不稳定，大量股票无基本面分 | ⭐⭐ |
| **无组合优化** | 选出Top-N后等权配置，无风险平价或均值-方差优化 | ⭐ |

## 三、升级方案（分三阶段，渐进式）

### 阶段一：扩因子 + 多策略（核心，2-3周）

**目标**：从单一5因子规则评分，升级为多套子策略并行打分 + 融合排序

#### 3.1.1 新增因子（从 ~10 个扩展到 ~40 个）

```
新增量价因子（基于现有OHLCV数据，零成本）：
├── 趋势强度类
│   ├── ADX（平均趋向指数，区分趋势/震荡市）
│   ├── 均线距离（MA5偏离MA20的百分比）
│   └── 新高新低比（N日新高/N日新低）
├── 动量反转类
│   ├── 短期反转因子（5日反转，A股特有有效）
│   ├── 多周期动量（10日、20日、60日分层）
│   └── 动量加速度（20日动量 - 10日动量，二阶导数）
├── 波动率类
│   ├── ATR（真实波幅，衡量活跃度）
│   ├── 历史波动率（20日年化波动率）
│   ├── 偏度（收益分布不对称性，正偏度好）
│   └── 下行波动率（只计下跌日的波动）
├── 成交量类
│   ├── 换手率（现有数据但未用于评分）
│   ├── 量价相关性（量价背离/共振信号）
│   ├── OBV背离（能量潮与价格的背离）
│   └── 缩量止跌（成交量萎缩+价格企稳）
└── 路径类
    ├── 路径依赖（最高价到当前的回撤幅度）
    └── 连续涨跌天数（过度延伸信号）
```

#### 3.1.2 多套子策略并行（核心创新点）

当前只有一套"5因子综合评分"，升级为 **4套独立子策略** 并行打分：

```
策略A：趋势跟踪策略（权重30%）
├── 适用：单边上涨/下跌市
├── 因子：ADX趋势强度 + MA排列 + MACD + 中期趋势
├── 逻辑：选择趋势明确、均线多头、MACD金叉的股票
└── 输出：0-100 趋势评分

策略B：动量反转策略（权重25%）
├── 适用：震荡市、A股特有的反转效应
├── 因子：短期反转(5日) + 多周期动量 + 动量加速度 + RSI
├── 逻辑：短期超跌反弹 + 中期趋势向上 = 最佳买点
└── 输出：0-100 动量评分

策略C：量价共振策略（权重25%）
├── 适用：放量突破、缩量止跌等关键节点
├── 因子：量比 + 换手率 + 量价相关性 + OBV + 缩量止跌
├── 逻辑：量在价先，寻找量价共振/背离信号
└── 输出：0-100 量价评分

策略D：低波动质量策略（权重20%）
├── 适用：防御性配置、熊市抗跌
├── 因子：波动率 + 偏度 + 下行波动 + 基本面(PE/PB/ROE)
├── 逻辑：低波动 + 合理估值 + 正偏度 = 稳健标的
└── 输出：0-100 质量评分
```

#### 3.1.3 策略融合机制

```
最终评分 = 策略A × 30% + 策略B × 25% + 策略C × 25% + 策略D × 20%
```

优势：
- 单策略失效时其他策略兜底
- 趋势市策略A主导，震荡市策略B+C主导，熊市策略D主导
- 多维度覆盖，不会因为单一信号错失好股票

#### 3.1.4 市场状态自适应（可选，阶段一后期）

```
市场状态识别（基于全市场涨跌比、指数MA位置）：
├── 牛市（指数>MA20 且 上涨家数>60%）
│   └── 动态调整权重：策略A:40%, B:25%, C:20%, D:15%
├── 震荡市（指数在MA20附近 ±3%）
│   └── 动态调整权重：策略A:20%, B:35%, C:30%, D:15%
└── 熊市（指数<MA20 且 上涨家数<40%）
    └── 动态调整权重：策略A:15%, B:15%, C:20%, D:50%
```

### 阶段二：行业维度 + 资金流向（进阶，2-3周）

#### 3.2.1 行业/板块因子

```
行业分类（申万一级31个行业）：
├── 行业内相对强度（股票涨幅 vs 行业涨幅）
├── 行业动量（行业指数过去N日涨幅排名）
├── 行业轮动信号（强势行业 + 行业内强势股 = 双重确认）
└── 行业中性化（因子值在行业内标准化，消除行业偏差）
```

#### 3.2.2 资金流向因子（需额外数据源）

```
北向资金（沪/深股通）：
├── 北向持仓占比变化（连续增持信号）
├── 北向资金净流入（日/周/月维度）
└── 北向持仓集中度

主力资金：
├── 主力净流入（大单动向，DDX/DDY）
├── 龙虎榜机构买入占比
└── 融资余额变化（杠杆资金态度）
```

#### 3.2.3 事件驱动因子

```
├── 业绩预告超预期（净利润 > 一致预期）
├── 机构调研密度（近期被调研次数）
├── 分析师评级上调
└── 限售股解禁（负面事件，短期抛压）
```

### 阶段三：AI增强 + 组合优化（远期，3-4周）

#### 3.3.1 机器学习排序

```
方案一：LightGBM排序模型
├── 特征：~40个因子 + 行业编码
├── 标签：未来N日超额收益（相对行业/指数）
├── 训练：滚动窗口训练（过去2年数据）
├── 预测：对候选股票打分排序
└── 优势：自动学习因子间非线性关系，比线性加权更优

方案二：因子IC加权（不引入ML，更稳健）
├── 每月计算各因子IC（信息系数）
├── 按IC绝对值动态分配因子权重
├── IC不显著的因子自动降权
└── 优势：可解释性强，不容易过拟合
```

#### 3.3.2 组合优化

```
当前：Top-N等权配置
优化后：
├── 风险平价（各股票对组合风险贡献相等）
├── 最大分散度（最大化组合分散化比率）
├── 最小方差（给定收益下最小化组合方差）
└── 约束条件：单票上限20%、行业上限40%、换手率限制
```

#### 3.3.3 完整Pipeline

```
数据准备 → 因子计算(40+) → 行业中性化 → 4策略并行打分
                                              ↓
                                        策略融合/ML排序
                                              ↓
                                        组合优化(权重分配)
                                              ↓
                                        风控过滤 → 最终组合
```

## 五、风险与注意事项

1. **过拟合风险**：因子越多不一定越好，需通过IC/IR检验因子有效性，淘汰无效因子
2. **调参陷阱**：每个策略都有超参，阶段一优先用业界通用参数，不要过度优化
3. **回测验证**：每新增一个模块都要在回测中验证边际贡献（新模块是否提升了夏普/降低了回撤）
4. **计算性能**：因子从10个到40个，遍历全市场5000+股票，需要关注耗时
5. **实盘可行性**：资金流向、龙虎榜等数据有延迟，需确认数据源的时效性
